# Geometry

## 概要

2次元・3次元空間における幾何学的計算を行うユーティリティクラスを提供します（旧 MathFunctions/GeometryUtils から分割）。

## 実装クラス

### [GeometryUtils](../../../src/lib/math/GeometryUtils.java)

- **用途**：
	- 線分・長方形の交差判定、点と図形の位置関係判定、各種距離計算
- **特徴**：
	- 線分の交差判定（2次元・3次元）
	- 点が線分上 / 図形内にあるかの判定
	- ユークリッド距離・マンハッタン距離・チェビシェフ距離の計算
- **時間計算量**：
	- 各クエリ $O(1)$
- **空間計算量**：
	- $O(1)$

## アルゴリズム（データ構造）選択ガイド

- 衝突判定や最近傍判定など、計算幾何の基礎演算が必要な問題で利用します。

## 注意事項

- 浮動小数点演算による誤差に注意してください。
