# Combinatorics

## 概要

組み合わせ論関連の計算を行うユーティリティクラスを提供します（旧 MathFunctions/CombinatoricsUtils から分割）。

## 実装クラス

### [CombinatoricsUtils](../../../src/lib/math/CombinatoricsUtils.java)

- **用途**：
	- 組み合わせ数 (nCr) ・順列数 (nPr) ・重複組み合わせ (nHr) などの計算
- **特徴**：
	- 通常版とモジュロ版の両方を提供
	- スターリング数（第2種）・ベル数の計算に対応
- **時間計算量**：
	- 前計算 $O(N)$ / クエリ $O(1)$（テーブル利用時）
- **空間計算量**：
	- $O(N)$（前計算テーブル）

## アルゴリズム（データ構造）選択ガイド

- 組み合わせ・順列の数え上げが必要な数え上げDP・確率計算で利用します。

## 注意事項

- モジュロ版を使う場合は事前に階乗・逆元テーブルの初期化が必要です（[Factorial](../Factorial)モジュール参照）。
