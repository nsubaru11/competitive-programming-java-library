# NumberTheory

## 概要

数論関連の計算を行うユーティリティクラスを提供します（旧 MathFunctions/NumberTheoryUtils から分割）。

## 実装クラス

### [NumberTheoryUtils](../../../src/lib/math/NumberTheoryUtils.java)

- **用途**：
	- 最大公約数 (GCD) ・最小公倍数 (LCM) ・拡張ユークリッドアルゴリズム・オイラーのトーシェント関数などの計算
- **特徴**：
	- 通常のGCDに加え、ビット演算を用いた高速版 (`gcd`) を提供
	- 拡張ユークリッドアルゴリズムによるモジュロ逆数の計算
	- 素数判定・素因数分解
- **時間計算量**：
	- $O(\log \min(a, b))$（GCD/LCM/exGCD）
- **空間計算量**：
	- $O(1)$

## アルゴリズム（データ構造）選択ガイド

- 整数論的な性質（互いに素・周期性など）を扱う問題で利用します。
- 素数判定・篩は[PrimeNumber](../PrimeNumber)モジュールも参照してください。

## 注意事項

- 入力値が0の場合の挙動は各メソッドのJavaDocを参照してください。
