# FastPrinter 利用ガイド

## 概要

`FastPrinter` は、競技プログラミング向けに設計された高速出力クラスです。  
内部バッファと低レベルなバイト操作を利用して `OutputStream` への書き込みを効率化し、標準の `PrintStream`
より小さいオーバーヘッドで出力できます。
`FastIO24` 版では Java 24 の `VarHandle` を用いた最適化を採用し、`ContestPrinter` / `CompressedFastPrinter` 系の機能も統合しています。

## 特徴

- **効率的な出力**: 既定で 1 MiB の内部バッファを利用して高速に出力します。
- **高速性の重視**: Java 24 版では `VarHandle` を利用し、数値書き込みや配列コピー経路を最適化しています。
- **メソッドチェーン**: `print()` / `println()` は自身を返すため、連結して記述できます。
- **多機能出力**: プリミティブ配列、オブジェクト配列、`Iterable`、2 次元配列、反転出力、繰り返し出力を扱えます。
- **関数変換**: 出力時に変換関数を適用できます。
- **`boolean` 出力**: `true/false` ではなく `Yes/No` を出力します。
- **容量管理**: バッファ不足時は一度フラッシュし、1 回の書き込みサイズが現バッファを超える場合のみ拡張します。
- **文字コード**: `ASCII` 範囲内の文字出力を前提としています。

## 依存関係

- Java 標準の入出力ライブラリ（`OutputStream`, `FileDescriptor`）
- `java.lang.invoke.VarHandle`
- `java.util.function.*`

## 主な機能

### 基本的な出力メソッド

| メソッド               | 説明                |
|--------------------|-------------------|
| `print(value)`     | 指定した値を改行なしで出力します。 |
| `println()`        | 改行のみを出力します。       |
| `println(value)`   | 指定した値を改行付きで出力します。 |
| `printf(fmt, ...)` | フォーマット付き出力を行います。  |

### 配列・コレクション出力

| メソッド                                        | 説明                        |
|---------------------------------------------|---------------------------|
| `print(int[] arr)`                          | `int[]` を空白区切りで出力します。     |
| `println(int[] arr)`                        | `int[]` を各要素改行区切りで出力します。  |
| `println(int[][] arr2d)`                    | 2 次元 `int` 配列を行列形式で出力します。 |
| `print(Iterable<T> iter)`                   | `Iterable` を空白区切りで出力します。  |
| `print(U[] arr, char delimiter)`            | 指定区切り文字で配列を出力します。         |
| `printReverse(...)` / `printlnReverse(...)` | 配列やコレクションを逆順で出力します。       |
| `printRepeat(...)` / `printlnRepeat(...)`   | 文字列や文字の繰り返し出力を行います。       |

### 関数変換付き出力

| メソッド                                          | 説明                      |
|-----------------------------------------------|-------------------------|
| `print(int[] arr, IntFunction<T> function)`   | 各要素を関数で変換して空白区切りで出力します。 |
| `println(Iterable<T> iter, Function<T, U> f)` | 各要素を関数で変換して改行区切りで出力します。 |

### 制御メソッド

| メソッド      | 説明                              |
|-----------|---------------------------------|
| `flush()` | 内部バッファの内容を強制的に書き込みます。           |
| `close()` | バッファをフラッシュした後、保持中の出力ストリームを閉じます。 |

## 利用例

```java
try (FastPrinter fp = new FastPrinter()) {
	int[] a = {1, 2, 3};
	fp.println("Result:").print(a).println();
	fp.print(10).print(' ').println(20);
	fp.println(true);
	fp.printRepeat('-', 10).println();

	List<Integer> list = List.of(4, 5, 6);
	fp.print(list, x -> x * 2).println();
}
```

## 注意事項

- `ASCII` 範囲外の文字はサポートしていません。
- `autoFlush=false`（デフォルト）の場合、最後に `flush()` または `close()` を呼ぶ必要があります。
- `close()` は内部で保持している出力ストリームを閉じます。既定コンストラクタでも標準出力を再利用する前提のコードでは呼び出しタイミングに注意してください。
- 統合に伴い、以前の `ContestPrinter` は削除されました。

### バージョン管理について

バージョン番号は 2 桁で管理します：

- 1 桁目（メジャーバージョン）: メソッドの追加や機能拡張があった場合に更新
- 2 桁目（マイナーバージョン）: 誤字修正、バグ修正、マイクロ高速化などの小さな更新があった場合に更新
